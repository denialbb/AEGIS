# `load-handoff` Skill

This skill automates the process of loading a handoff file generated by an external agent (such as OpenCode) and integrating its context, goals, and tasks into the current session.

## When to Use This Skill
Use this skill at the beginning of a session when a handoff document has been generated by another agent to resume work.

## Instructions

When the user asks you to load a handoff or runs `/load-handoff`:

1. **Run the Handoff Loader**: Execute the `load_handoff.py` script:
   ```bash
   wsl -d Arch .venv/bin/python .agents/scripts/load_handoff.py
   ```

2. **Understand the Handoff Context**: Read the printed content of the handoff file. Pay close attention to:
   - The primary objective and goals.
   - The files modified or targeted in the previous session.
   - The "suggested skills" to invoke.

3. **Initialize the Tasks**: Use the details from the handoff to initialize the local `task.md` checklist. 

4. **Align Workspace State**: Verify which branch is checked out and ensure that you align with the expected starting state.
